# Khancafe: Entrenador de sumas en Python

Este proyecto es un pequeño entrenador interactivo de matemáticas escrito en Python.

El programa genera sumas aleatorias de dos dígitos y le pide al usuario que responda.
Si la respuesta es correcta, lleva un conteo de cuántas respuestas correctas se han hecho de forma seguida.
Cuando el usuario logra tres respuestas correctas consecutivas, el programa lo felicita por haber dominado las sumas.
